## Launcher fix

首次尝试的错误为：

```text
Error: Cannot find module 'D:\A8'
```

原因是 `Start-Process -ArgumentList` 没有把含空格的完整 `server.js` 路径作为单一参数。`scripts/start-gallery.ps1` 现先规范化入口路径，再显式传入带引号的单一参数，同时保持以下机制不变：

- Node 继续通过 `-NodePath`、启动器进程的 `NODE_EXE` 或 PATH 选择；
- 环境变量仍由现有公共脚本注入；
- stdout/stderr 仍写入 Runtime 日志；
- JSON PID 元数据字段和停止前核对方式不变。

修复前先以 `node --check` 对同一个含空格入口路径做非服务型参数边界测试，退出码为 0。
