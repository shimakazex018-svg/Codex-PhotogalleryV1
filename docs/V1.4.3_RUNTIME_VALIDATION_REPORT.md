## Failure

`start-gallery.ps1` 使用 `Start-Process -ArgumentList` 传递项目中的 `server.js` 路径。项目路径含空格，但参数没有作为一个受保护的完整参数传给 Node。Node 因而尝试加载截断路径 `D:\A8` 并退出。

stderr 核心错误：

```text
Error: Cannot find module 'D:\A8'
code: 'MODULE_NOT_FOUND'
Node.js v24.18.0
```

这是启动脚本参数转义缺陷，不是数据库、媒体路径、端口或 Node 版本故障。
