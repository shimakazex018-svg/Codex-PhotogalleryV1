# V1.2 Cleanup Report

## Scope and rules

本报告记录 V1.2 的目录、文档、脚本和核心代码静态审计。分类含义：

- A：确认删除。必须无引用、无路由、无动态调用、无配置依赖且不影响启动/API。
- B：归档。有历史价值，但已被当前维护文档替代或包含过期事实。
- C：保留。当前生效、条件生效、用途不确定，或缺少运行级证据。

本阶段未启动网站，不修改端口、媒体/数据目录、数据库、API、页面逻辑或核心业务文件。
